# Review_Scrapper-AWS

A Python-based web application that scrapes product reviews from e-commerce websites and is designed for deployment on AWS. The project uses Flask for the web interface and BeautifulSoup/Requests for web scraping.

## Features
* Scrape product reviews from online shopping websites
* Simple web interface to enter product details
* Logs scraping activity for debugging
* AWS deployment ready
* Lightweight and easy to modify

## Technologies Used
* **Python**
* **Flask**
* **Requests**
* **BeautifulSoup**
* **HTML & CSS**
* **AWS**

##  Project Structure

review_scrapper-aws/
